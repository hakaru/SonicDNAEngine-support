# SonicDNA Engine サポート

SonicDNA Engine サポートページへようこそ。

SonicDNA Engineは、IR/NAMギターアンプシミュレーターアプリで、AUv3プラグインにも対応しています。

---

## 🎯 概要

SonicDNA Engineは、iPhoneやiPadをギターアンプ＆キャビネットシミュレーターに変えます。IR（インパルスレスポンス）ファイルでキャビネットシミュレーション、NAM（Neural Amp Modeler）ファイルでアンプモデリングを行い、リアルタイムの低レイテンシーオーディオ処理を実現します。

---

## ✨ 主な機能

### 🎸 NAMアンプモデリング
Neural Amp Modelerファイルを読み込んで、AI駆動のデジタルモデリングで実際のアンプの音を再現します。

### 🔊 IRキャビネットシミュレーション
インパルスレスポンスファイルを読み込んで、FFTベースの畳み込みによる正確なキャビネットシミュレーションを行います。

### 🎛️ 内蔵エフェクト
- **ノイズゲート** - 調整可能なスレッショルドとスムーズなゲーティング
- **3バンドEQ** - ローシェルフ、ミッドベル、ハイシェルフのパラメトリックEQ
- **入出力ゲイン** - 信号レベルの完全な制御
- **Dry/Wetミックス** - 原音とエフェクト音のブレンド

### 🔌 AUv3プラグイン
AUv3対応ホストアプリでオーディオユニットエクステンションとして使用：
- AUM、GarageBand、Cubasis、BeatMakerなど
- ホストアプリ内でのフルパラメータコントロール
- プラグイン内からIR/NAMファイルの読み込み

### 🎵 スタンドアロンエフェクターモード
USBオーディオインターフェース経由でギターを接続し、スタンドアロンのアンプシミュレーターとして使用：
- 低レイテンシーのリアルタイムオーディオ処理
- USBオーディオインターフェース対応
- 48kHzサンプルレート

### 💾 プリセットシステム
IR/NAMの選択とすべてのパラメータ値を含む、お気に入りの設定を保存・呼び出し。

## 🔌 セットアップガイド

### スタンドアロンモード

```
ギター → オーディオI/F → iPhone/iPad → オーディオI/F → アンプ/ヘッドホン
                          (SonicDNA Engine)
```

1. USBオーディオインターフェースをiPhone/iPadに接続
2. ギターをオーディオI/Fの入力に接続
3. オーディオI/Fの出力をアンプ/ヘッドホンに接続
4. SonicDNA Engineを起動
5. 「Start Audio Engine」をタップ
6. IRおよび/またはNAMファイルを読み込み

### AUv3プラグインモード

1. AUv3ホストアプリ（AUM、GarageBandなど）を開く
2. SonicDNA Engineをオーディオユニットエフェクトとして追加
3. プラグインインターフェースからIR/NAMファイルを読み込み
4. 必要に応じてパラメータを調整

## 📱 アプリの使い方

### IRファイルの読み込み

1. **IR**タブに移動
2. **+** をタップしてWAVファイルをインポート
3. IRファイルを選択してアクティブに
4. 対応フォーマット: WAV（任意のサンプルレート、モノ/ステレオ）

### NAMファイルの読み込み

1. **NAM**タブに移動
2. **+** をタップして.nam/.jsonファイルをインポート
3. NAMモデルを選択してアクティブに
4. 対応: WaveNet、LSTM、Linearアーキテクチャ

### シグナルチェーン

```
入力 → 入力ゲイン → ノイズゲート → NAM → EQ → IR畳み込み → 出力レベル → ミックス → 出力
```

### コントロール

| コントロール | 範囲 | 説明 |
|-------------|------|------|
| **INPUT** | 0 - 2倍 | 処理前の入力ゲイン |
| **OUTPUT** | 0 - 2倍 | 処理後の出力レベル |
| **MIX** | 0% - 100% | Dry/Wetブレンド（0%=原音、100%=エフェクト音） |
| **ノイズゲート** | -96 〜 0 dB | ゲートスレッショルド |
| **EQ Low** | -12 〜 +12 dB | ローシェルフ（200 Hz） |
| **EQ Mid** | -12 〜 +12 dB | ミッドベル（1 kHz） |
| **EQ High** | -12 〜 +12 dB | ハイシェルフ（4 kHz） |

## ❓ よくある質問

### どのオーディオインターフェースに対応していますか？
クラスコンプライアントなUSBオーディオインターフェースであれば動作するはずです。ZOOM AMS-22などの一般的なインターフェースでテスト済み。

### IRファイルはどこで入手できますか？
IRファイルは多くのソースから入手可能です：
- [SonicDNA Collector](https://apps.apple.com/app/id6758269736)で自分で作成
- ToneHuntなどのコミュニティからダウンロード
- 多くの無料・有料IRパックがオンラインで入手可能

### NAMファイルはどこで入手できますか？
NAMモデルは以下で見つかります：
- [ToneHunt](https://tonehunt.net) - コミュニティNAMモデル共有
- [Neural Amp Modeler](https://github.com/sdatkinson/neural-amp-modeler)を使って自分でトレーニング

### どのNAMアーキテクチャに対応していますか？
SonicDNA EngineはWaveNet、LSTM、LinearのNAMアーキテクチャに対応しています。

### レイテンシーはありますか？
アプリは48kHzで128サンプルバッファ（約2.7ms）を使用しています。全体のラウンドトリップレイテンシーはオーディオインターフェースによります。

### GarageBandで使えますか？
はい！SonicDNA EngineはGarageBandやその他のAUv3ホストでAUv3エフェクトプラグインとして動作します。

### SonicDNA Collectorとの関係は？
SonicDNA Collectorはアナログ機器の音響特性をキャプチャします。SonicDNA Engineはそのキャプチャされた特性を再生します。2つのアプリでSonicDNAエコシステムを構成します：キャプチャ → 再生。
